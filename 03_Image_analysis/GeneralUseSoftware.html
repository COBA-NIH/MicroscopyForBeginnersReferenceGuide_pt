Software de uso geral#

As ferramentas nesta página tendem a ser relativamente polivalentes em vários tipos de análises e/ou imagens. Para ferramentas especializadas em determinadas etapas de análise, consulte a página Software de uso específico.

ImageJ#

ImageJ 27,28,29 é um programa de processamento de imagens capaz de operar em uma variedade de imagens, incluindo multicanal, 3D e séries temporais. Ele fornece uma variedade de operações básicas de processamento de imagem, mas pode ser complementado com uma variedade de plug-ins para tarefas mais complexas.

Quais são suas desvantagens?

  • As operações de processamento de imagem são feitas uma de cada vez, embora tenha a capacidade de processar em lote e criar macros, mas requer algum conhecimento de linguagem de programação.

  • Mas, embora possa abrir imagens grandes, há um limite de tamanho para manipular as imagens com base na memória disponível. E mesmo que consiga abrir imagens grandes, isto pode diminuir o desempenho.

Como baixar/instalar e saber mais?

Para baixar o ImageJ ou sua distribuição Fiji com “plugins incluidas”, acesse

Para documentação e tutoriais sobre como usar o ImageJ, bem como uma lista de plugins disponíveis

CellProfiler#

CellProfiler 30 é um software desenvolvido para biólogos por biólogos; ele cria uma ponte entre a análise de imagem e o cientista sem a necessidade de conhecimento computacional. Ele foi projetado com a ideia de um pipeline de análise de imagem em mente: permite que você use uma série de módulos interoperáveis para projetar seu próprio pipeline de análise personalizado que pode ser aplicado a uma ou milhares de imagens, tornando-o adequado para análise de imagem de alto rendimento .

Quais são suas desvantagens?

  • O CellProfiler não pode lidar com imagens grandes, como seções inteiras de tecido de experimentos histológicos. O tamanho da imagem é atualmente limitado pela memória disponível em seu computador.

  • Embora o CellProfiler possa realizar análises em imagens 3D, a visualização é limitada a um plano z por vez por meio de um controle deslizante na janela de visualização.

  • Além disso, vários recursos do CellProfiler estão disponíveis apenas para imagens 2D.

QuPath#

QuPath 31 oferece um amplo conjunto de ferramentas de análise de imagem que podem ser aplicadas a imagens de lâminas inteiras. Por isso é muito utilizado com imagens de patologias, mas também pode ser utilizado com outras imagens. O QuPath também contém ferramentas de classificação de pixels e pode ser integrado ao ImageJ (por exemplo, para enviar ROIs entre os programas ou para acessar plug-ins do ImageJ).

Quais são suas desvantagens?

  • Para obter o máximo do QuPath (especialmente ao analisar muitas imagens), é necessário codigos básicos e algum conhecimento de linguagem de programação (ou adaptação do código de outras pessoas).

Icy#

Icy 32 é uma ferramenta de análise de imagem pronta para uso, que utiliza plugins para criar protocolos visuais de análise de imagem que podem ser compartilhados com outros usuários.

Quais são suas desvantagens?

  • A interoperabilidade do Icy com outros softwares é limitada ao ImageJ

MIB (Navegador de imagens de microscopia)#

MIB 33 é um software amigável para análise de imagens de conjuntos de dados multidimensionais para microscopia de luz e eletrônica. Ele permite que você use todos os dados adquiridos para sua análise e extração de características morfológicas.

Quais são suas desvantagens?

  • Foi criado usando o MATLAB, existe uma versão independente, mas ela não usa a versão mais atualizada do MATLAB

napari#

napari 34 está sendo desenvolvido como um visualizador de imagens multidimensional que pode ser expandido por meio de uma variedade de plug-ins para executar tarefas básicas e complexas de análise de imagens.

Quais são suas desvantagens?

  • O napari ainda está em fase de desenvolvimento, mas é uma plataforma bastante popular e já possui diversos plugins e casos de uso com tutoriais disponíveis.
