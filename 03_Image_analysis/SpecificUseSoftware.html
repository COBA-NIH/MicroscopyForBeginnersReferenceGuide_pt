Software de Uso Específico#

As ferramentas nesta página tendem a ser extremamente boas em determinadas tarefas, mas são menos destinadas a usos mais comuns. Para ferramentas com áreas de foco mais amplas, consulte a página Software de uso geral.

Cellpose#

Cellpose 35 é um algoritmo de segmentação, que fornece uma interface gráfica do usuário em que permite aos usuários usar modelos treinados ou treinar seus próprios usando suas imagens e anotações.

Em que tipo de problema de análise de imagem este software é melhor?

Segmentação de objeto, a maioria dos modelos treinados são para a segmentação de células, mas podem ser aplicados para segmentar outros objetos semelhantes

Quais são suas desvantagens?

  • A sua utilização requer algum conhecimento computacional.

  • Treinar um novo modelo requer correção da anotação manual que pode ser demorada, mas provavelmente consome menos tempo do que outros métodos de treinamento.

  • 🎥 Como usar o tutorial

ilastik#

ilastik 36 é uma ferramenta para classificação interativa de imagens, segmentação e análise. Ele aproveita os algoritmos de aprendizado de máquina para executar a classificação em nível de pixel e objeto. Usá-lo não requer experiência em processamento de imagem.

Em que tipo de problema de análise de imagem este software é melhor?

Ele pode ser usado tanto para a segmentação em instancia quanto para a segmentação semântica. Ele também executa segmentação e rastreamento, embora com um pouco menos de parâmetros ajustáveis do que algumas outras ferramentas disponíveis.

Quais são suas desvantagens?

  • Às vezes, carregar ou exportar imagens pode exigir um pouco de paciência para obter as dimensões corretas.

  • O ilastik é limitado pela RAM do seu computador, portanto, treinar um modelo com muitos recursos ou trabalhar com imagens muito grandes provavelmente o deixará lento.

Piximi#

Piximi é um aplicativo para anotação e classificação que é executado inteiramente no seu navegador e não requer instalação e configuração mínima.

Em que tipo de problema de análise de imagem este software é melhor?

Piximi pode fazer classificação de imagens usando aprendizado de máquina

Quais são suas desvantagens?

  • Ainda está em fase de desenvolvimento e alguns dos recursos propostos ainda não estão disponíveis
